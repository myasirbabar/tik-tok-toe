# Artificial Intelligence Tik-Tok-Toe
Tik Toc Toe with AI build in C++. It is a console game.

---------------

### How to Begin
- Open and run source file in compiler

### How To Play
- It is one / two player game.
- If You are single player then AI will play with you. 
- User Friendly interface
